# APOSTILLATES & NOTARY SERVICES Website

A modern, responsive one-page website for APOSTILLATES & NOTARY SERVICES located in Arizona. Built with React and TailwindCSS, featuring a clean, professional design with blue and gold accents.

## Features

- **Modern Design**: Clean, minimalistic design with professional aesthetics
- **Responsive**: Mobile-first design that works perfectly on all devices
- **Interactive**: Smooth scroll navigation and hover effects
- **Professional**: Blue and gold color scheme with modern typography
- **Fast**: Optimized for performance and user experience

## Sections

1. **Hero Section**: Eye-catching headline with call-to-action
2. **Services**: Grid layout showcasing all services with icons
3. **Why Choose Us**: Features and benefits with statistics
4. **Contact**: Contact form and business information
5. **Footer**: Business details and social media links

## Technologies Used

- React 18
- TailwindCSS 3
- Modern CSS animations
- Responsive design principles
- Google Fonts (Poppins & Inter)

## Getting Started

### Prerequisites

- Node.js (version 14 or higher)
- npm or yarn

### Installation

1. Clone or download the project files
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) to view it in the browser

### Building for Production

To create a production build:

```bash
npm run build
```

This builds the app for production to the `build` folder.

## Customization

### Colors
The color scheme can be customized in `tailwind.config.js`:
- Primary Blue: `#1e40af`
- Secondary Blue: `#3b82f6`
- Accent Gold: `#f59e0b`
- Light Gold: `#fbbf24`

### Content
Update the content in the respective component files:
- `src/components/Hero.js` - Hero section content
- `src/components/Services.js` - Services list and descriptions
- `src/components/WhyChooseUs.js` - Features and statistics
- `src/components/Contact.js` - Contact information and form
- `src/components/Footer.js` - Footer content and links
